# amazon-bedrock-text-generation-apps
This project demonstrates AI-powered text generation using Amazon Bedrock and the Amazon Nova Lite model with Python and Boto3.

Features
1) AI-generated poem
2) Pet recipe generator
3) Pet adoption description generator

Technologies Used
1)Python
2) Amazon Bedrock
3) Amazon Nova Lite
4) Boto3
5) JSON

Project Files
1) task2.py – Text generation example
2) task3.py – AI-powered pet recipe generator
3) task4.py – AI-powered pet adoption description generator
